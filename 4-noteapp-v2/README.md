# noteapp

##笔记APP(Vue + Vuex + localStorage)

 老王笔记，改变3-noteapp中删除一条记录后使用emit更新父级列表的方式，使用Vuex，方便管理更大型的SPA

###介绍

* 使用npm run dev，然后访问http://localhost:8080可以及时预览修改的内容
* 使用npm run build打包
* 手机访问[Demo](http://jaliy.github.io/noteapp/4-noteapp-v2/)

###项目地址：

```shell
git clone https://github.com/jaliy/noteapp.git
```
